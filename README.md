# reverserLang
Программа на случай, если вы набрали текст не на той раскладке.

1. Если при запуске программы вместо нормального текста отображается какая-то краказябра,
то необходимо сменить шрифт консоли на любой TrueType (Это, например, Lucida Console)

2. (Объяснение на примере) При выборе программы нужно иметь ввиду, что она считает, что
при наборе символов .,? вы не меняете раскладки.
